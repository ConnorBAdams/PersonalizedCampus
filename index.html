Project Goals

TailoredTours was built in 3 sprints throughout the Fall 2020 semester. The goals of each sprint are detailed below.

Sprint 1 Goals: We first began with setting up a very basic website using the Inovatik template with the end goal of showcases the project and its features. Secondly, we needed to determine the framework for the project and then have each team member learn it. This made sure we took the steps needed to prepare and plan for the application, such as wireframing. The login and account creation of the app was the primary hurdle to jump here as an account it reuired before the rest of the application can be used. We decided to offer Google login for low barrier usage. During this sprint, we decided to focus on the "tour creation" side of the application. We laid down the foundation for the next two sprints by building tours with routes and location nodes.


Sprint 2 Goals: For this sprint, we decided to focus heavily on the tour creation side of the application to allow users to customize more aspects of the tour besides the route itself, such as pictures and a sharing feature. The ability to accept multiple routes was included to allow the user to have more customized routes to take. We added in the tour edit screen so that the user can edit all aspects of the tour, from nodes to routes and naming. Locatin nodes features were increased to allow things such as pictures and videos to be included at each node should the creator want to add one.


Sprint 3 Goals: The final sprint was more focused on the tour taking side now that the tour making was more solid. The user has the option of taking the tour either in person, using a map, or virtually, using a carousel of location nodes. Tour sharing became much simpler as a QR code is now generated through the app that can be shared and scanned to start the tour for a user. The application received a nice polish on most screens to improve user functionality and in general make it look more put together. The completion of this website and the demo video linked above was the final task of this sprint.

About TailoredTours & CS495

The idea of TailoredTours came about while taking the course CS495 at the University of Alabama. The class is primarily focused on group work to build an application. The original goal was to make a campus tour app but evolved to an app that works for any location.
